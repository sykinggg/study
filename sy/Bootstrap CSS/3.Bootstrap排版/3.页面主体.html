页面主体 基本思想:Bootstrap 将全局 font-size 设置为 14px，line-height 设置为 1.428。 这些属性直接赋予 元素和所有段落元素。 另外，

（段落）元素还被设置了等于 1/2 行高（即 10px）的底部外边距（margin）。 中心内容 基本思想:通过添加 .lead 类可以让段落突出显示。 使用 Less 工具构建 基本概念:variables.less 文件中定义的两个 Less 变量决定了排版尺寸：@font-size-base 和 @line-height-base。 第一个变量定义了全局 font-size 基准 第二个变量是 line-height 基准 注意:使用这些变量和一些简单的公式计算出其它所有页面元素的 margin、 padding 和 line-height.
