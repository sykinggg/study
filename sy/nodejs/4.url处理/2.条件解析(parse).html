parse函数 —— 条件解析 基本概念:parse函数的第二个参数是布尔类型，当参数为true时，会将查询条件也解析成json格式的对象。 代码实现: 运行结果: { protocol: 'http:', slashes: true, auth: null, host: 'www.baidu.com', port: null, hostname: 'www.baidu.com', hash: null, search: '?page=1', query: { page: '1' }, pathname: '/', path: '/?page=1', href: 'http://www.baidu.com/?page=1' }
